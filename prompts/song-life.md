# Task: 中文歌词评价

## Context
用户分享原创歌词，希望获得专业、客观的评价反馈。评价需要从多个维度分析，既肯定亮点也指出可改进之处。

## Instructions

1. **通读全文**：理解歌词的主题、情感基调和叙事结构

2. **分析亮点**：
   - 立意/主题是否独特或有深度
   - 结构是否清晰（verse/chorus/bridge）
   - 情感表达是否真诚、不矫情
   - 意象和用词是否有画面感

3. **提出观察**：
   - 押韵和节奏感
   - 是否过度依赖旋律
   - 用词密度和自然度
   - 副歌的记忆点

4. **给出整体印象**：
   - 完成度评价
   - 歌词与旋律/演唱的配合预期

5. **互动**：询问创作背景或灵感来源

## Expected Outcome

提供一份结构化的歌词评价，包含亮点、建议和整体印象，语气客观友好，避免空泛的夸奖或过度批评。
